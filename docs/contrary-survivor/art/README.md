# Арты ContrarySurvivor — входящая папка

Сюда Ринат кладёт сгенерированные арты (по промптам из `GDD.md`, Приложение E).
game-lead просматривает и вставляет ссылки в GDD.

**Формат:** PNG / JPG / WEBP (любой из них читается напрямую).

**Желательные имена** (чтобы я сразу понял, что есть что; если несколько вариантов — добавляй суффикс `-1`, `-2`):
- `hero.png` — главный герой (промпт 1)
- `bandit.png` — бандит (промпт 2)
- `village.png` — окрестности деревни (промпт 3)
- `styleframe.png` — стиль-кадр (промпт 4)

Если имена не по шаблону — не страшно, просто скажи мне, что положил, и я посмотрю.
